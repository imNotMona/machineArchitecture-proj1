1st project of machine architecure course
Though "Python" is shown as the main language, those are for the tests used for grading written code
my written code is in the following files:
gradebook_main.c
gradebook.c

can be ran through the terminal after compiling with the make file
info for make is within file, but pasted here for convenience
> make                          # build all programs'
> make clean                    # remove all compiled items'
> make clean-tests              # remove temporary testing files'
> make zip                      # create a zip file for submission'
> make test                     # run all tests'
> make test testnum=n           # run only test number n'

REQUIRES to be ran on linux, or a linux container
